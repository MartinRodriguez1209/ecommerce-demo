# ecommerce-demo

Este proyecto es una demostración de un front-end de e-commerce especializado en la venta de hardware y componentes informáticos.

Tecnologías: Desarrollado con HTML, CSS utilizando el framework Bootstrap y JavaScript para la lógica y la interactividad del lado del cliente.

Funcionalidades Clave:

Manejo de Usuarios: Incluye la simulación de flujos de autenticación y autorización (login/registro).

Administración de Comercio: Presenta una interfaz específica (área de administrador/vendedor) para la creación, modificación y eliminación de productos del catálogo (CRUD simulado).

E-commerce Core: Catálogo de productos, carrito de compras y gestión básica de pedidos (simulada).

Simulación de Entorno RESTful: La interacción de la aplicación con la API se realiza mediante peticiones HTTP. Para el desarrollo, se utilizaron:

JSON Server: Empleado para simular un backend RESTful funcional con persistencia de datos.

Postman: Utilizado para probar y validar manualmente las peticiones y respuestas HTTP.
